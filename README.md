# Alexander - Shoe Store Android APP
============

Buy shoes for yourself in our store.
---

## Features
- View binding
- Data binding
- Live data
---

## Implementation
- MVVC pattern(usage of Binder)
- Directories naming reflect MVVC pattern data/model, viewModel and views
